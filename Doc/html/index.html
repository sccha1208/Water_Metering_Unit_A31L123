ABOV Cortex-M0+ Core based MCUs Integrated Driver
ABOV F/W Package API Documentation

Introduction

  • This ABOV F/W package provides API documentation for all the peripherals in the ABOV 32-bit MCUs Driver Library. This driver library provides an easy to use API for developing applications for the ABOV 32-bit MCU devices based ARM Cortex-M core.
  • Common driver, example, third party libraries in the /common directory.
  • Each device's specific drivers are provided in the /device directory.
